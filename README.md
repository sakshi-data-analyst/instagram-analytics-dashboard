# instagram-analytics-dashboard

## Project Overview
Developed an Instagram Analytics Dashboard to analyze use engagement, likes , comment , follwers , and account  performance.

## Tools Used
- MySQL
- Power BI
- Excel

## Key Metrics
- Total Likes
- Total Comments
- Engagement Rate
- Follower Growth
- Top Performing Posts

## Features
- Interactive Dashboard
- KPI Cards
- Trend Analysis
- User Engagement Insights

## Outcome
Provided actionable insights to understand audience behavior and improve social media performance
